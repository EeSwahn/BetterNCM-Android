# BNA 源码文件说明

本文按“可读源码/配置文件”维度整理当前项目中每个文件的作用，便于快速定位功能。

说明：`jks/jar/webp/png` 等二进制文件（如签名文件、图片资源、Gradle Wrapper JAR）不属于源码文本文件，故不在下表逐项展开。

## 根目录与构建系统

- `build.gradle.kts`：项目级 Gradle 构建入口，声明插件但不直接应用到 root。
- `settings.gradle.kts`：配置插件仓库、依赖仓库、根工程名，并包含 `:app` 模块。
- `gradle.properties`：项目级 Gradle 参数（JVM、AndroidX、R class 策略等）。
- `gradle/libs.versions.toml`：版本目录，统一管理依赖版本与插件别名。
- `gradle/wrapper/gradle-wrapper.properties`：Gradle Wrapper 分发地址、校验与下载配置。
- `gradlew`：Unix/macOS 下的 Gradle Wrapper 启动脚本。
- `gradlew.bat`：Windows 下的 Gradle Wrapper 启动脚本。

## App 模块构建与清单

- `app/build.gradle.kts`：Android 应用模块构建配置（SDK、签名、Compose、依赖等）。
- `app/proguard-rules.pro`：混淆/压缩规则模板（当前主要保留默认内容）。
- `app/src/main/AndroidManifest.xml`：应用清单，声明权限、`MainActivity` 与 `MusicService`。

## 数据模型层

- `app/src/main/java/com/example/bna/data/model/Models.kt`：集中定义 API 数据模型（登录、搜索、歌曲、歌词、二维码等）。

## 网络与加密层

- `app/src/main/java/com/example/bna/data/network/ApiClient.kt`：统一 Retrofit/OkHttp 客户端；管理 Cookie、匿名 Token、身份伪装与请求拦截。
- `app/src/main/java/com/example/bna/data/network/NeteaseApiService.kt`：网易云接口定义（登录、搜索、播放地址、歌词、推荐等）。
- `app/src/main/java/com/example/bna/data/network/LenientGsonConverterFactory.kt`：自定义 Gson Converter，兼容非严格 JSON 与流消费问题。
- `app/src/main/java/com/example/bna/data/network/AnonymousToken.kt`：生成并拉取匿名 `MUSIC_A` Token，补齐未登录请求能力。
- `app/src/main/java/com/example/bna/data/network/SuiXinChangApi.kt`：独立封装“随心唱伴奏”接口请求与 eapi 加密流程。
- `app/src/main/java/com/example/bna/util/NeteaseCrypto.kt`：网易云 `weapi/eapi` 加密核心工具（AES/RSA/MD5）。
- `app/src/main/assets/netease_api/crypto_bundled.js`：打包后的 JS 加密实现（weapi/eapi/linuxapi），供兼容或对照使用。

## 仓储层

- `app/src/main/java/com/example/bna/data/repository/MusicRepository.kt`：业务数据仓库；封装登录、二维码、搜索、播放 URL、推荐、歌词等请求与错误处理。

## 播放与音频处理

- `app/src/main/java/com/example/bna/player/MusicPlayer.kt`：全局播放器核心，管理 ExoPlayer、播放队列、模式切换、进度同步与状态流。
- `app/src/main/java/com/example/bna/player/SuiXinChangAudioProcessor.kt`：自定义音频处理器，混合伴奏与人声通道并控制人声音量。
- `app/src/main/java/com/example/bna/service/MusicService.kt`：前台播放服务，创建通知渠道并托管播放器生命周期。

## ViewModel 层

- `app/src/main/java/com/example/bna/viewmodel/LoginViewModel.kt`：登录页状态管理（验证码登录、二维码登录、轮询与会话判断）。
- `app/src/main/java/com/example/bna/viewmodel/SearchViewModel.kt`：搜索页状态管理（查询、播放、退出登录事件）。
- `app/src/main/java/com/example/bna/viewmodel/LyricsViewModel.kt`：歌词状态管理（加载、行定位、LRC/YRC 解析、随心唱控制）。
- `app/src/main/java/com/example/bna/viewmodel/SettingsViewModel.kt`：歌词动画设置状态管理（开关与参数更新）。

## 应用入口与导航

- `app/src/main/java/com/example/bna/MainActivity.kt`：应用入口 Activity，初始化 `ApiClient`、启动 `MusicService`、挂载导航。

## UI 主题与动画

- `app/src/main/java/com/example/bna/ui/theme/Color.kt`：项目主题色与语义色定义。
- `app/src/main/java/com/example/bna/ui/theme/Type.kt`：Material3 字体排版体系定义。
- `app/src/main/java/com/example/bna/ui/theme/Theme.kt`：Compose 主题组装（暗色方案 + Typography）。
- `app/src/main/java/com/example/bna/ui/animation/LyricsAnimations.kt`：歌词 UI 动效 Modifier 与动画配置模型。

## UI 页面（screen）

- `app/src/main/java/com/example/bna/ui/screen/SplashScreen.kt`：启动页动画与登录态检查后的路由跳转。
- `app/src/main/java/com/example/bna/ui/screen/LoginScreen.kt`：登录页面 UI（扫码/验证码两种流程与错误展示）。
- `app/src/main/java/com/example/bna/ui/screen/MainScreen.kt`：主容器页，组合搜索页、底部播放条与歌词页弹出。
- `app/src/main/java/com/example/bna/ui/screen/SearchScreen.kt`：歌曲搜索主界面（查询、结果列表、下一首、退出登录）。
- `app/src/main/java/com/example/bna/ui/screen/PlayerBar.kt`：底部迷你播放条（当前歌曲、进度、播放/暂停、清空队列）。
- `app/src/main/java/com/example/bna/ui/screen/LyricsScreen.kt`：歌词全屏承载页，按设备切换平板/手机歌词布局。
- `app/src/main/java/com/example/bna/ui/screen/LyricsSettingsScreen.kt`：独立歌词动画设置页（开关与恢复默认）。

## 歌词子模块（screen/lyrics）

- `app/src/main/java/com/example/bna/ui/screen/lyrics/PhoneLyricsLayout.kt`：手机歌词页布局（封面/歌词双页、自动隐藏控件、底部设置弹层）。
- `app/src/main/java/com/example/bna/ui/screen/lyrics/TabletLyricsLayout.kt`：平板歌词页布局（左右分栏、布局校准参数、设置弹层）。
- `app/src/main/java/com/example/bna/ui/screen/lyrics/LyricsComponents.kt`：歌词渲染核心组件（滚动追焦、逐字高亮、翻译行与点击跳转）。
- `app/src/main/java/com/example/bna/ui/screen/lyrics/LyricsPlaybackComponents.kt`：歌词页播放控制组件（进度条、播放按钮、播放模式、播放列表弹层）。
- `app/src/main/java/com/example/bna/ui/screen/lyrics/LyricsControlWidgets.kt`：歌词页辅助控件（随心唱按钮、参数滑块、设置 BottomSheet）。
- `app/src/main/java/com/example/bna/ui/screen/lyrics/LyricsUtils.kt`：歌词工具函数（偏好存取、时间格式化、逐字进度与兜底字信息生成）。

## 资源 XML

- `app/src/main/res/values/strings.xml`：字符串资源（当前主要是应用名）。
- `app/src/main/res/values/colors.xml`：基础颜色资源（模板色）。
- `app/src/main/res/values/themes.xml`：系统主题声明（`Theme.BNA`）。
- `app/src/main/res/xml/backup_rules.xml`：自动备份规则模板。
- `app/src/main/res/xml/data_extraction_rules.xml`：数据提取/迁移规则模板。
- `app/src/main/res/drawable/ic_launcher_background.xml`：启动图标背景矢量图。
- `app/src/main/res/drawable/ic_launcher_foreground.xml`：启动图标前景矢量图。
- `app/src/main/res/mipmap-anydpi/ic_launcher.xml`：自适应启动图标声明。
- `app/src/main/res/mipmap-anydpi/ic_launcher_round.xml`：圆形自适应启动图标声明。

## 测试代码

- `app/src/test/java/com/example/bna/ExampleUnitTest.kt`：本地 JVM 单元测试模板。
- `app/src/androidTest/java/com/example/bna/ExampleInstrumentedTest.kt`：Android 仪器测试模板。
